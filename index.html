Key Features

  • ✦ Animated progress rings & streaks

02

Notes App

A minimal, Firebase-powered note-taking experience.

Overview

Built for focus and simplicity, this notes app syncs in real-time via Firebase. The UI strips away distractions so you can capture thoughts instantly, then organise them with tags and folders.

Key Features

  • ✦ Real-time Firebase Firestore sync
  • ✦ Rich text editing (bold, italic, lists)
  • ✦ Color-coded note categories
  • ✦ Search & filter functionality
  • ✦ Offline-first architecture

Tools Used
